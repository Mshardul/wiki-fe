# Learning Path: Standard SWE

FAANG-style interview bar. Core data structures, the interview patterns built on them, and the algorithms a standard SWE loop actually probes — no CP-only structures, no disk/systems internals. ~14-16 weeks at a steady pace.

See [DSA Learning Paths](../index.md#learning-paths) for the other tracks.

## Explicitly skipped in this track

Suffix array, skip list, segment tree, fenwick tree, B-tree, B+-tree, bloom filter, interval tree, full AVL/Red-Black internals, network flow, string hashing/matching beyond the basics, number theory, bit manipulation, advanced DP patterns (bitmask/interval/state-machine). Covered in [Senior-depth](./senior-depth.md) or [CP-weighted](./cp-weighted.md) where relevant.
