# Learning Path: CP-weighted

Competitive-programming leaning. CP-primitives and contest-frequent structures get priority — number theory, range structures, string algorithms, combinatorial search — over general interview practice problems. ~14-16 weeks at a steady pace.

See [DSA Learning Paths](../index.md#learning-paths) for the other tracks.
